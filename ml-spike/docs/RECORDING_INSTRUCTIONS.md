# Recording Instructions for Barn Camera

Send these instructions to someone at the barn.

## What to Record

Using the **HikConnect app** on your phone:

1. Open HikConnect app
2. Tap on the camera
3. Tap the **Record** button (red circle)
4. **Record 2-3 minutes** of a milking session
5. Tap **Stop** when done

## What to Capture

Try to record a **complete milking process** if possible:
- Person approaching the cow
- Spraying / pre-cleaning
- Stripping
- Attaching milking machine
- Milking in progress
- Detaching machine
- Post-dip application

If you can't get a full session, just record whatever is happening.

## How to Send the File

1. In HikConnect app, go to **Album** or **Gallery**
2. Find the recorded clip
3. Tap **Share** → send via WhatsApp/email to the developer

## Tips

- Record during **daytime** if possible (better lighting)
- Make sure the camera shows both cow positions
- Don't move the phone — just record the screen

## After Sending

The developer will process the video and test the detection system.
