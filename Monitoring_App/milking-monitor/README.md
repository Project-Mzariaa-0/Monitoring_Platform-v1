# Milking Process Compliance Monitor

Monorepo scaffold for the web app, inference service, and shared contracts described in `agent_plans/application/app.json`.

## Layout

- `apps/web/` Next.js supervisor app
- `apps/inference-service/` Python RTSP inference service
- `packages/shared-contracts/` JSON schemas shared by both apps
- `docs/` planning docs, ADRs, and implementation notes

This repository currently contains structure and starter files only.
